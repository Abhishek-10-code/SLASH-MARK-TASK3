# Weather Forecast Website

This repository contains the code for a simple weather forecast website that allows users to check the weather for the current day and the next five days. The site uses an API to fetch real-time weather data and offers both manual location input and automatic location detection features.

## Features

- **Weather Forecast:** Get the weather forecast for the current day and the next five days.
- **Manual Location Input:** Users can manually enter their location to retrieve weather data.
- **Automatic Location Detection:** The app can automatically detect the user's location and display the weather forecast for that location.
- **Responsive Design:** The website is responsive and works well on different devices.

## Technologies Used

- **HTML5**

## Usage

1. **Enter a Location:**
   - Type in a city name or other location in the input box and click "Get Weather" to fetch the weather forecast for the specified location.

2. **Automatic Location Detection:**
   - Click "Detect Location" to automatically fetch the weather forecast for your current location.

## API Key Setup

This project uses the OpenWeatherMap API to fetch weather data. To set it up:

1. Sign up for a free API key at [OpenWeatherMap](https://openweathermap.org/).

2. Open the `script.js` file and replace the placeholder `YOUR_API_KEY` with your actual API key:

   ```javascript
   const apiKey = 'YOUR_API_KEY'; // Replace with your OpenWeatherMap API key
   ```

3. Save the changes and reload the website.
